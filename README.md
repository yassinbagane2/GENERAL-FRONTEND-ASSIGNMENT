To start the app:

-   Create a database for this project and make sure to update the ".env" file accordingly

-   Run the following commands:

*   composer install
*   npm install
*   npm run dev
*   php artisan migrate
*   php artisan serve

Used packages: 
    - Flowbite for design : "https://flowbite.com/"
    - GSAP for animations: "https://gsap.com/"
